Local build note
================

Do NOT use `make build` on this machine with Python 3.14.

The upstream requirements.txt pins Pillow==11.0.0. On this system that version tries to compile from source and fails with missing Python.h / JPEG headers.

Use the local compatible requirements file instead:

    cd /home/carlos/DSS/InsecureWebApp
    ./run-local.sh

This keeps requirements.txt unchanged for the academic project, but uses requirements-local.txt locally with:

    pillow>=12.0.0

The application will start at:

    http://127.0.0.1:5000
